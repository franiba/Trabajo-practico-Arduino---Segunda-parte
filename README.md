# Trabajo-practico-Arduino---Segunda-parte
Este repositorio consta de los codigos realizados en processing y arduino los cuales estan comunicados por el monitor serie, permiten realizar las siguientes acciones
* Desde el hardware, cada boton encendera un led designado por arduino
* En processing se recibira dicha accion y mostrara en pantalla cual fue el led que se acciono
* A su vez se podra controlar ambos leds desde botones  ubicados en la pantalla de processing
